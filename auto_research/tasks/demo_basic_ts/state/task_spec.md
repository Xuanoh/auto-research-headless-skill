# Demo BasicTS AutoResearch Task

## Objective

Demonstrate the AutoResearch file protocol on a BasicTS-oriented task without changing model code.

## Workspace

/path/to/BasicTS

## Success Criteria

- The task can build a state pack.
- The task can run a smoke experiment command.
- The task can parse a metric and record an iteration.

## Constraints

- Use read-only or smoke commands for the demo.
- Do not modify BasicTS source files in this demo task.

## Initial Direction

Validate the protocol machinery before launching real model experiments.
